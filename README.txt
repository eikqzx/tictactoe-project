1.Unzip env.zip to root project folder.
2.Open terminal and type
```bash
npm install --legacy-peer-deps.
```
3.Run the development server by type
```bash
npm run dev
```
in terminal
Open http://localhost:3000 in browser to see the web tic tac toe game. Or go to https://tictactoe-game-teal.vercel.app for online production
